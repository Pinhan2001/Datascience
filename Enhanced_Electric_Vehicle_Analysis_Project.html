Electric Vehicle Trends and Insights: A Data-Driven Analysis

Introduction

Electric vehicles (EVs) are transforming the transportation industry, driven by the need for cleaner energy and sustainable solutions. This project analyzes trends in electric vehicles using real-world data, uncovering insights into adoption patterns, manufacturer dominance, and performance characteristics.

Objectives

  • Explore the distribution of EV types (Battery Electric Vehicles and Plug-in Hybrid Electric Vehicles).
  • Analyze EV adoption patterns by geographical location.
  • Understand EV adoption trends over time and across manufacturers.
  • Assess EV performance trends, such as electric range.

Dataset Overview

The dataset contains detailed information about electric vehicles, including:

  • Registration locations (county, city, and state).
  • Vehicle specifications (make, model, year, and electric range).
  • EV type: Battery Electric Vehicle (BEV) or Plug-in Hybrid Electric Vehicle (PHEV).

Key Statistics:

  • Number of Records: 71721
  • Number of Columns: 17
  • Date Range: Includes model years spanning several decades.

Data Cleaning

Before conducting analysis, it's important to clean the data. This includes:

  • Checking for missing values.
  • Handling inconsistent data entries.
  • Dropping irrelevant columns if necessary.

# Import required libraries
import pandas as pd
import matplotlib.pyplot as plt

# Load the dataset
file_path = 'Electric_Vehicle_Population_Data.csv'
df = pd.read_csv(file_path)

# Check for missing values
missing_values = df.isnull().sum()
print("Missing values in each column:")
print(missing_values)

# Drop rows with missing data in critical columns
df = df.dropna(subset=['Electric Vehicle Type', 'Model Year', 'Make'])

# Fill missing values in non-critical columns if necessary (e.g., location data)
df['County'] = df['County'].fillna('Unknown')

# Ensure all Model Year values are integers
df['Model Year'] = df['Model Year'].astype(int)

# Verify the cleaned data
print("\nData after cleaning:")
print(df.info())

# Import required libraries
import pandas as pd
import matplotlib.pyplot as plt
# Analyze trends in EV types
ev_type_counts = df['Electric Vehicle Type'].value_counts()

# Plot the distribution of electric vehicle types
plt.figure(figsize=(8, 6))
ev_type_counts.plot(kind='bar', color=['skyblue', 'lightgreen'], edgecolor='black')
plt.title('Distribution of Electric Vehicle Types', fontsize=16)
plt.xlabel('Electric Vehicle Type', fontsize=12)
plt.ylabel('Number of Vehicles', fontsize=12)
plt.xticks(rotation=45, fontsize=10)
plt.grid(axis='y', linestyle='--', alpha=0.7)
plt.show()

import pandas as pd
import matplotlib.pyplot as plt
# Group data by model year and count the number of vehicles
yearly_trends = df.groupby('Model Year')['Electric Vehicle Type'].value_counts().unstack()

# Plot the trends over time
plt.figure(figsize=(10, 6))
yearly_trends.plot(kind='line', marker='o', ax=plt.gca())

plt.ylabel('Number of Vehicles', fontsize=12)
plt.legend(title='EV Type', fontsize=10)

Insights and Conclusions

Key Findings:

  1. Electric Vehicle Types:
    • Battery Electric Vehicles (BEVs) dominate the market compared to Plug-in Hybrid Electric Vehicles (PHEVs).
  2. Trends Over Time:
    • EV adoption has significantly increased over the past decade, with BEVs showing faster growth.
  3. Geographical Distribution:
    • Certain counties have higher EV registrations, indicating possible infrastructure advantages or population density.
  4. Manufacturer Analysis:
    • Tesla leads the EV market, followed by other major manufacturers like Nissan and Chevrolet.
